# Architecture Overview

## Frontend
- Next.js App Router
- TypeScript
- Premium component system with motion and feedback layers

## Backend Platform
- Supabase Postgres + RLS
- Auth + Storage + Realtime-ready schemas
- Server actions for secure business operations

## AI Layer
- OpenAI chat and embeddings
- Usage logging and cost tracking
- Semantic search via pgvector function

## SaaS Model
- Multi-tenant organizations and role-based memberships
- Billing/subscription-ready schema
- Admin/audit/security settings surfaces
