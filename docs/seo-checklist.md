# SEO Checklist

## Metadata
- Title template and description configured
- OG and Twitter cards configured
- Canonical URL configured

## Crawlability
- `sitemap.xml` route active
- `robots.txt` route active

## Structured Data
- Organization
- WebSite
- SoftwareApplication
- Product
- FAQPage
- BreadcrumbList
- WebPage

## Content
- H1 hierarchy and semantic sections
- FAQ and trust sections present
- Internal links to legal/docs pages
